# SampleForCocos

Sample for Cocos2d (Games)

## description

Cocos2d-x のサンプルプロジェクト  
ライセンスとかはとくにないので、  
きになるものがあれば、ご自由に

## サンプル一覧(シーン別)

### MenuSample

Menuのおしっぱ判定サンプル  
`AnzMenu`(Menuを継承したもの)使用

### SoundMenuSample

Menuの一括効果音設定サンプル
`AnzMenu`(Menuを継承したもの)使用
